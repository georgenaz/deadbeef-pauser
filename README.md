deadbeef-pauser
===============

Sets pause for deadbeef  on activation of screenlock/screensaver.



Description
--------------

Скрипт запускается в фоне и при активации screensaver'а/блокировке экрана ставит на паузу deadbeef, если он запущен.



Requirements
--------------

- python2.7
- GNOME / KDE
- *linux*
